🏎️ Dashboard Operacional - L.A Custom (Matriz & Factory)
Este projeto consiste em um dashboard interativo desenvolvido para a gestão e monitoramento das operações de estética automotiva de alto padrão da L.A Custom, operando dentro da Porsche Stuttgart. O sistema centraliza dados de fluxo de veículos, performance de consultores e tipos de serviços realizados em duas unidades distintas: Matriz e Factory.

📊 Funcionalidades Principais
O dashboard foi estruturado com um sistema de abas para permitir uma visão clara de cada setor da operação:

1. Unidade Matriz (Stuttgart)
Focada no volume e fluxo contínuo de lavagens técnicas e atendimento direto:

Controle de Fluxo: Monitoramento de volume semanal (ex: 73 veículos na última semana).

Ranking de Consultoria: Acompanhamento de produtividade por consultor (Gabriel, Nelson, Alexandre, Rafael).

Mix de Modelos: Distribuição detalhada dos modelos Porsche atendidos (Liderança de Cayenne e 911).

2. Unidade Factory (Galpão)
Focada em serviços de alta complexidade e proteção automotiva:

Gestão de Serviços Premium: Acompanhamento de aplicações de PPF, Coating, Blindagens (BSS) e polimentos técnicos.

Responsabilidade Técnica: Monitoramento das autorizações e entregas lideradas pelo setor técnico (Thiago Paixão).

🛠️ Tecnologias Utilizadas
Este projeto faz parte da iniciativa "Projeto Agenda", que visa substituir processos manuais por fluxos digitais eficientes:

HTML5/CSS3: Estrutura e estilização com interface premium (Dark Mode) adaptada ao padrão de luxo automotivo.

JavaScript & Chart.js: Gráficos dinâmicos para visualização de dados em tempo real.

GitHub Pages: Hospedagem para acesso rápido via web/mobile durante as operações em campo.

Data Analysis: Processamento de dados via Python/Pandas para consolidação de relatórios mensais.
